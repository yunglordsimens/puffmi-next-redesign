## / DESIGN DIRECTION

The visual identity takes cues from premium consumer electronics and streetwear brands â€” clean, high-contrast, product-first.

* **Aesthetic:** Minimal / Editorial with bold typography
* **Palette:** White base with saturated product color accents
* **Typography:**
    * **Headings:** Uppercase, tight tracking â€” impact-first
    * **Body:** Clean sans-serif for UI clarity
* **User Experience:** Scrolling storytelling, instant color feedback, smooth cart flow

## / KEY IMPLEMENTATIONS

1. **Animated Hero:** GSAP-powered entrance sequence with vape product as central visual
2. **Dynamic Color Selector:** Product image updates instantly on color pick â€” no page reload
3. **Marquee Component:** Smooth infinite scroll brand ribbon
4. **Collection Grid:** Responsive bento-style layout with product category cards
5. **Cart Notifications:** Sonner toast system for add-to-cart feedback
